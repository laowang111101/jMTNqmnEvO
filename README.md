# 旅游景点推荐系统 python38

## 项目概述

本项目是基于Spark和Django的旅游景点推荐系统，结合大数据分析与可视化技术，为用户提供个性化的旅游路线规划和景点推荐。系统通过采集去哪儿等旅游网站的数据，运用多种推荐算法，实现智能推荐功能，并通过大数据运算进行销量分析和省份热评分析。

## 技术栈

- **后端语言与框架**：Python、Django
- **数据库**：MySQL
- **大数据处理**：Spark、Hadoop、Hive
- **网络爬虫技术**：requests
- **机器学习算法**：Spark MLlib
- **前端技术**：BootStrap、ECharts

## 功能模块

### 用户管理

- 用户登录注册

### 推荐系统

- 旅游路线规划
- 多种推荐算法（如景点推荐、协同过滤推荐、基于内容的协同推荐）

### 大数据分析

- Spark MLlib、Spark ALS运算分析
- 大屏可视化，包括景点销量分析和省份热评分析

### 数据处理与展示

- 利用snownlp进行好差评分析
- pandas数据分析
- ECharts数据可视化

## 系统角色

- 用户：可登录注册，获取个性化的旅游推荐
- 管理员：管理用户数据，监控系统运行状态

- Python运行环境
